# 서울안녕내과 대시보드

환자 진료 데이터 시각화 대시보드. Google Sheets 데이터를 Streamlit으로 시각화.

## 페이지 구성

1. **환자정보** — KPI, 전년 동기 비교, 월간 성장률, 내원 추이, 요일×시간대 히트맵, 환자 지도
2. **지역장악도** — 행정동·연령대별 인구 대비 환자 비율 (시장 침투율)
3. **마케팅성과분석** — 캠페인 전후 성과 비교, 신환 트렌드

## 실행

```bash
streamlit run 1_환자정보.py
```

## 배포

Streamlit Cloud (메인 파일: `1_환자정보.py`)
